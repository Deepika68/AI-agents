# AI Research Agent with Prompt Engineering and Retrieval-Augmented Generation

## Overview

This project implements a modular AI Research Agent capable of retrieving real-time information from the web and generating context-aware responses using Large Language Models (LLMs).

The system combines DuckDuckGo search with prompt engineering techniques to perform Retrieval-Augmented Generation (RAG), enabling the model to answer user queries based on external knowledge.

## Features

- Web Search Integration using DuckDuckGo

- Prompt Engineering

- Retrieval-Augmented Generation (RAG)

- LLM Integration (Gemini/Groq)
